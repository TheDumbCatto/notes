# Important Properties

- $a \oplus b \oplus a = b$

Think of the first XOR in the above equation as message $b$ encrypted with key $a$, and the second XOR as the decryption.

- $(a \oplus b) \oplus c \leftrightarrow a \oplus (b \oplus c)$
